# AI-Powered Mock Interview Platform

## Problem
Students lack realistic interview practice, objective feedback, and adaptive difficulty during mock interviews.

## Solution
An AI-powered interview platform that simulates real interviews by:
- Analyzing resume and job description
- Asking adaptive interview questions
- Enforcing time pressure
- Scoring objectively
- Generating readiness reports

## Features
- Resume & JD analysis
- Easy → Medium → Hard questions
- Dynamic difficulty adaptation
- Time-based evaluation
- Early interview termination
- Final interview readiness score

## Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Python, Flask
- AI Logic: Rule-based (Extensible to LLMs)
